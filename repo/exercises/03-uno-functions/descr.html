Interactions between cards

Follow-up cards

When it's your turn in UNO, here's what usually happens. First try to find a suitable card from your hand and play it. If you have no suitable cards, then you pick one up from the draw pile. If that card is suitable, then you can play it. Otherwise, you simply hold on to it and play moves to the next player.

I've emphasized the word "suitable" on purpose. This suitability is in fact a relation between cards! In particular, you can add a card to the discard pile if it meets any of the following conditions.

  • The value of the top card of the discard pile equals the value of the card you want to play; e.g. a 3 of any color can be played on top of a 3 of any color.
  • The color of the top card of the discard pile equals the color of the card you want to play; e.g. any green card can follow any other green card.

Let's call this relation between cards the can_follow relation. What we need to implement is a function that decides this relation. That is, implement the function can_follow : card -> card -> bool that outputs true if and only if the second card can be played on top of the first card.
